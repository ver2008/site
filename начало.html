На этом сайте Вы сможете познакомиться с основами программирования на HTML и CSS. Но для начала давайте разберемся что это за языки и для чего они нужны.

HTML

HTML - язык разметки текста, используется для обработки текста и чтобы придать тексту определенный размер или выбрать стиль текста.

CSS

CSS - язык, который используют для структурирования текста, чтобы задать цвет и размер определенному элементу, в основном используют вместе с HTML.

Теперь когда Вы знаете для чего нужны эти языки можно приступать к изучению. Чтобы перейти на страницу с материалами и заданиями кликните по кнопке HTML или CSS в зависимости от того какой язык Вы хотите изучать.
